如果看到错误代码 12803：

这可能是由于 CloudStudio 平台的重定向配置错误导致的。建议的解决方案：

  1. 立即方案：在阿里云 DNS 设置中，暂时将域名的 CNAME 记录指向一个可以正常访问的地址（例如 GitHub Pages 提供的地址）。
  2. 长期方案：将网站迁移到 GitHub Pages（免费、支持自定义域名、HTTPS 自动证书）。

迁移步骤：

  • 1. 在 GitHub 上创建新仓库（例如 `chengong-website`）
  • 2. 将本地的网站文件（index.html 等）上传到仓库
  • 3. 在仓库设置中启用 GitHub Pages，选择分支（例如 main）
  • 4. 在 GitHub Pages 设置中添加自定义域名 `liulianghuo.com`
  • 5. 在阿里云 DNS 中，将域名的 CNAME 记录指向 `你的用户名.github.io`
  • 6. 等待 DNS 生效（通常几分钟到几小时）
